Workshop Topics

This workshop focuses on deep learning applied to large-scale unstructured data, such as that found in social media, healthcare, IoT devices, and online reviews. The international event will bring together academic and industrial researchers to exchange cutting edge research enabling Large-scale Unsupervised learning. Deep Learning to perform in a reliable and user-independent manner handling uncertainty and unforeseen events.

The topics of interest include, but are not limited to the following:

Deep learning:
  • Data Cleaning, Analysis Algorithms
  • Data Storage and Management
  • Privacy Preservation and Security Issues in Big Data
  • Data Compressionn
  • Machine Learning
  • Deep learning with big or small data
  • Deep Recognition Models
  • Deep Generative Models
  • Parallel and distributed computing techniques
  • Visualization methods for the analysis of large-scale data
  • Machine Learning in Natural Language Processing
  • Applications of Natural Language Processing
